Vertical Align

What does vertical align do?

The vertical-align CSS property specifies the vertical alignment of an inline or table-cell box. In order for this to work, the elements need to be set alone a baseline. As in, inline (e.g. <span>, <img>) or inline-block (e.g. as set by the display property) elements.

vertical-align is used to align inline-level elements. These are elements, whose display property evaluates to inline, inline-block or inline-table (not considered in this article).
