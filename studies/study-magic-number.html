Magic Number

Nov 10 2015
CSS Tricks - Magic Numbers in CSS
JS is involved (magic-number-css.js)

Introduction

  • Magic numbers is a programming term for "unnamed numerical constant"
  • It's difficult to understand what these numbers are for, without being familiar with it
  • In CSS, it refers to value which work under certain circumstances but are prone to break

Dropdown

source

In this dropdown example, the dropdown layer has to be position below its parent. Hence, the css will look something like this

    ul {
        display: none;
        position: absolute;
        top: 30px;
        left: 0px;
    }

top: 30px is a magic number. The only reason that it works is because its parent is also 30px tall. 30px is entirely circumstantial. If the font size of parent is changed, the height of the li will change as well.

If Chrome render it as 30px but firefox render it as 31px, then this method would not work. Hence the suggested solution is using top: 100%;

    ul {
        display: none;
        position: absolute;
        top: 100%;
        left: 0px;
    }
