CSS Positioning

1st Dec 2015
Source

Relative Position

The box went 10px to the left side because the position is set as relative

stuff above
Im divbox
stuff below 
.divbox {
    width: 150px;
    height: 150px;
    background-color: #BADA55;
    position: relative;
    left: 10px;
}

Static Position

If set as static then it won't move

stuff above
Im divbox
stuff below 
.divbox {
    width: 150px;
    height: 150px;
    background-color: #BADA55;
    position: static;
    left: 10px;
}

Static Position

Note that if the top is set as 50px, it'll be pushed down. However, notice that the position it occupied is still there. The stuff above and below wasn't pushed down.

stuff above
Im divbox
stuff below 
.divbox {
    width: 150px;
    height: 150px;
    background-color: #BADA55;
    position: relative;
    left: 10px;
    top: 50px
}

Absolute Position

Absolute removes element from the document flow

stuff above
Im divbox
stuff below 
.divbox {
    position: relative;

    span {
        position: absolute;
        bottom: 0px;
    }
}

Fixed Position

Absolute removes element from the document flow

stuff above
the div below me was fixed on top
Im divbox with fixed position

stuff below 
.divbox {
    position: fixed;
    z-index: 1000;
    width: 100%;
    padding: 20px;
    top: 0px;
    left: 0px;
    border: 5px solid red;
}

Z-index doesn't work on statically positioned objects

However, if you still need to use position static, and you need to overlay objects, you can use a margin-top: -10px to shift its position

setting the below will stretch the div to the whole page

div {
position: fixed;
top: 0;
bottom: 0;
left: 0;
right: 0;
background: red;
}

Hence if you need to cover the page but leave a 10px border on its surrounding

 div {
 position: fixed;
 top: 10px;
 bottom: 10px;
 left: 10px;
 right: 10px;
 background: red;
 }

This technique is suitable for creating modal boxes with dim background

div {
position: fixed;
top: 10px;
bottom: 10px;
left: 10px;
right: 10px;
background: rgba(0,0,0,0.5);
}
