CSS Animation

Intro to CSS Animation

Setting Linear on the animation will make it move linearly as oppose to easing

.city {
    height: 250px;
    background: url('../studies/study-css-animation-image/city.jpg');
    background-repeat: repeat-x;
    -webkit-animation: move-city 1s infinite linear;
}

/*Animation*/
$image-width: 533px;
@-webkit-keyframes move-city {
    0% {background-position: 0 0;}
    100% {background-position: -$image-width 0;}
}

Setting alternate on the animation will move it back and forth

#monkey {
    position: absolute;
    bottom: -35px;
    left: 0px;
    -webkit-animation: monkey-shake 1s infinite alternate linear;
    z-index: 1;
}

@-webkit-keyframes monkey-shake {
    from {left: 50px;}
    to { left: 75px;}
}

The second value in monkey-shake and monkey-enter is the delay before the animation sets in

#monkey2 {
    position: absolute;
    bottom: -35px;
    left: -1000px;
    z-index: 1;
    -webkit-animation: 
        monkey-shake 0.5s 2s infinite alternate,
        monkey-enter 1s 1s
}
/*Animation*/
@-webkit-keyframes monkey-shake {
    from {left: 50px;}
    to { left: 75px;}
}

@-webkit-keyframes monkey-enter {
    from {left: -1000px;}
    to {left: 50px;}
}
                
#copter {
    position: absolute;
    right: 0px;
    -webkit-animation: 
        copter-hover 1s infinite alternate linear;
}

@-webkit-keyframes copter-hover {
    from {top: 50px;}
    to {top: 60px;}
}
